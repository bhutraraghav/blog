My first experience as a web developer

Internship at Tamilnadu Science and Technology Centre

May 20, 2020

This summer I got an opportunity to work for the Tamilnadu Science and Technology centre. My project was to make a digital content web-application to be served on a kiosk at TNSTC for the topic of astronomy. I started by learning the front end development from scratch and over a span of 5 weeks, developed slowly and steadily the webpage. A portion of the developed webpage can be viewed here.

The project also required good graphical contents and Interactivity. We added videos, images and also a quiz for evaluating what the user has learnt. The quiz was developed using JavaScript so we made a two dimensional array which would contain questions, their options and reasoning and description of the Correct option.

We also loaded random set of questions every time somebody attempt a quiz so the questions would load from question bank which is a 2D JavaScript array and we use the random number generator function called crypto.getRandomValues it generates Random numbers in the 32 bit array and then we just take the modulo of all the values of of the random number array with the number of questions available in our question bank (the 2D array). And then we put the questions of the respective question number in order in the quiz.
